Inteligencia Artificial: El Futuro

Posted by León Solórzano on Sunday, September 5th 2021

All modern websites and web applications are built using three fundamental technologies: HTML, CSS and Javascript.
These are the languages of the web.

In this post, let's focus on HTML. We will learn what HTML is all about, and why you too should learn it.

What is HTML?

HTML stands for HyperText Makeup Language. It's a makeup language that web develpers use to structure and describe the content of a webpage (not a programming language).

HTML consists of elements that describe fifferent types of content: paragraphs, links, headings, images, video, etc. Web browsers understand HTML and render HTML code as websites.

In HTML, each element is made up of 3 parts:

  1. First element
  2. The opening Tag
  3. The closing Tag
  4. The actual element

You can learn more at MDN Web Docs.

Why should you learn HTML?

There are countless reasons for learning the fundamental language of the web. Here are 5 of them:

  • To be able to use the fundamental web dev language
  • To hand-craft beautiful websites instead of relying on tools like Worpress or Wix
  • To build web applications
  • To impress friends
  • To have fun 😃
